# Assets Directory Guide

Use this folder to store media used by the site.

## Recommended structure

- `assets/logo/`
  - Place brand logos and alternate logo variants here.
- `assets/photos/`
  - Place product photos, lifestyle images, and marketing visuals here.
- `assets/icons/`
  - Place UI icons, social icons, and favicon source files here.

## Notes

- Prefer web-optimized formats (`.svg`, `.webp`, `.png`) where possible.
- Keep file names descriptive and consistent (for example: `logo-primary.svg`, `toy-shape-sorter.webp`).
